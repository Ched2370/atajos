## Condicionales

### Zero condicional

La condicional cero se usa para hablar o explicar hechos generales. Es decir, cosas que son verdades universales. En este tipo de frases, se conocen las consecuencias de la condición.

- If + Present Simple + Present Simple
- “If that thing happens...that happens”

### First Conditional

El primer condicional se emplea cuando nos referimos a situaciones en las que es bastante probable que algo tenga lugar.

- If + Presente simple + ‘Will’ + Infinitivo
- “If this thing happens...that will happen”

### Second Conditional

El segundo condicional lo utilizamos para referirnos a situaciones en las que hay poca probabilidad de que algo ocurra, aunque aun así es posible.

- If + Pasado Simple + ‘Would’ + Infinitive
- “If this thing happened...that thing would happen / that thing would be happening”

### Third Conditional

El tercer condicional lo utilizamos cuando hablamos de situaciones que ya es imposible que tengan lugar, hubo una posibilidad en el pasado pero ya no.

- If + Pasado Perfecto + ‘Would’ + ‘Have’ + Participio Pasado
- “If this thing had happened...that thing would have happened / that thing would have been happening”
